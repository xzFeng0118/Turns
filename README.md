# Turns Marketplace (Expo + React Native + TypeScript)

## Getting started

1. Install dependencies

```sh
npm install
```

2. Run the app

```sh
npm run start
```

Then press:
- `a` for Android
- `i` for iOS
- `w` for web

## Structure

- `App.tsx` - App entry
- `src/navigation` - Stack + tab navigation
- `src/screens` - Placeholder screens
- `src/data` - Mock data for browsing items
